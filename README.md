# clgrep

grep for changelog.

## Install

`go get github.com/jiro4989/clgrep`

## Usage

`clgrep [options] search_word file_name`

### Options

```
Usage:
  main [OPTIONS]

Application Options:
  -i, --ignore-case   Ignore case
  -r, --reverse       Reverse sort
  -t, --tag           Search tags
      --today-format= Only today
      --show-indent   Show indent

Help Options:
  -h, --help          Show this help message
```

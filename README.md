# clgrep

grep for changelog.

## Install

`go get github.com/jiro4989/clgrep`

## Usage

`clgrep [options] search_word file_name`

### Options

- `-h` Help
- `-i` Ignore case
- `-r` Reverse sort
- `-t` Search tags
- `--today` Only today
- `--no-indent` Remove indent
